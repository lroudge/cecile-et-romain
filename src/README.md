## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Environment variables

You will need a Google Cloud Project API key to run this project in dev and see the embedded Google Maps iframe in the "directions" page, and store it in a `.env` file under the name `GOOGLE_API_TOKEN`.
